# canna-oss

**OSS platform for Brazilian patient cannabis associations — ANVISA RDC 1.014/2026 sandbox.**

Self-hosted first. AGPL-3.0 + CLA. LGPD per-member crypto. SNGPC nativo. Zero vendor lock-in em dados de saúde.

## Gap Confirmado

Nenhum produto existente cobre simultaneamente:

- **LGPD** per-member AES-256-GCM + crypto-deletion (Art. 18 IV)
- **SNGPC XML** nativo (homologado ANVISA)
- **BSPO** trimestral/anual auto-gerado
- **CPC 29 / IAS 41** ativo biológico
- **Self-hosted** — associação é controladora única
- **OSS auditável** — código aberto = trust moat para dados de saúde em setor cinzento

## Stack

`Next.js 15` · `Fastify 5` · `PostgreSQL 16` · `MinIO` · `Kamal 2` · `Redis + BullMQ` · `Drizzle ORM` · `TOTP sem SaaS auth`

## Contexto de Mercado

- 315 associações ativas no Brasil (operando sob HC judicial)
- 873k pacientes estimados ANVISA 2026
- Sandbox RDC 1.014 vigente agosto/2026 — edital esperado abr–jul/2026
- TAM R$130M → SAM R$47M → SOM R$27M (2030)
