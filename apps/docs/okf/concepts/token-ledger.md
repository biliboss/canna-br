# Contribuição é registrada num ledger sem especulação

**Assertiva:** o token do canna-br é uma unidade de contribuição/reputação num
ledger event-sourced auditável — não um ativo negociável. A infra é o negócio; a
associação não é.

O ledger é uma projeção sobre o stream do [domain-kernel](domain-kernel.md) e
sustenta o modelo de governança DAO descrito no negócio.
