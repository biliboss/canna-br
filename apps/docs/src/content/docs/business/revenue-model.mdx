> **Premissa inegociável:** organização saudável nos moldes O2, com remuneração competitiva **mundialmente** — todos os envolvidos prosperam. O preço é consequência do custo da dignidade do time, não do teto de dor do cliente.

## Modelo Base

```
Self-host (AGPL)          → Grátis para sempre
Managed hosting           → Pago (infra + suporte + SLA)
Professional services     → Limitado (hard limit: max 20% receita)
```

O código é livre, a conveniência é paga. Quem não pode pagar faz self-host — a associação paga pela conveniência, não pelo software.

## Reconciliação com a Infraeconomics

> Os números da tese [Infraeconomics](/business/tokenomics/) ({SITE.thesisCeiling.displayRevPerAssoc}; {SITE.thesisCeiling.displayRefRevenue}) são o **cenário-teto** de referência: associação madura consumindo todas as camadas A–F (marco v0.6–v0.8). A ponte entre chão e teto é o roadmap econômico: cada camada ativada (compliance, compras, equipamentos, crédito) move a associação do plano base em direção ao teto. Um número é o chão contratado, o outro é o teto da tese — complementares, não contraditórios.

## Travessia: O2 + CONTRIB (sem maquiagem)

> Remuneração diferida e auditável ≠ trabalho de graça. É o que separa organização saudável de exploração de entusiasmo.

- Entre v0.1 e ~{SITE.breakeven.assocLayersCE.high} associações pagantes **não há** receita para salário global. Vale realista: 12–24 meses.
- Durante o vale: cada hora vira posição CONTRIB no [ledger](/business/dao/), a **valor de mercado competitivo** (não descontado), pública e auditável.
- Quando a receita cruza o break-even, a cascata (waterfall) converte CONTRIB acumulado em pagamento real, em ordem transparente.

Princípios O2 embutidos:

| Princípio | Como |
|---|---|
| Transparência total | Remuneração pública no ledger |
| Distribuição justa | Por papel e contribuição |
| Sustentabilidade antes de retorno | Break-even primeiro, waterfall depois |

## FACT Deal

A FACT (Federação das Associações Cannabis Terapêutica) representa {SITE.factDeal.assocCount} associações membros. Um contrato federação permite:

```
Modelo B — Fixed + Per Member (preferido)
  R$ 1.500/mês fixo (suporte federação + dashboard consolidado)
  + R$ 8/membro/mês (média 40 membros/associação = R$ 320/assoc)
  36 associações × R$ 320 = R$ 11.520 + R$ 1.500 = R$ 13.020/mês → R$ 156.000 ARR
```

**{SITE.factDeal.modelLabel} alinha incentivos:** canna-br ganha quando a associação cresce em membros, e é mais previsível para a FACT (preço fixo de base).

O dashboard consolidado federação (view agregada across todas as {SITE.factDeal.assocCount} associações) é o diferencial do contrato — exclusivo do plano Rede/Enterprise.
