# Mercado Brasil — TAM / SAM / SOM

## Contexto

873k pacientes estimados pela ANVISA para 2026. 315 associações ativas operando sob HC judicial. Sandbox RDC 1.014 vigente agosto/2026 — janela que legitima e expande o setor.

## Tamanho de Mercado

| Nível | 2026 | 2028 | 2030 | Definição |
|---|---|---|---|---|
| **TAM** | R$25M | R$65M | R$130M | Todo software cannabis BR (pharma + B2G + associativo) |
| **SAM** | R$9M | R$22M | R$47M | Software para associações (315 → ~800 projeção) |
| **SOM** | R$1.8M | R$8M | R$27M | OSS managed hosting ~57% penetração até 2030 |

## Mercado Pharma Cannabis BR (contexto)

- TAM pharma: R$971M (2026) — inclui importação, farmácias, prescrição médica
- Crescimento CAGR estimado: 30–40% a.a. (pré-sandbox)
- Sandbox ANVISA = catalisador de crescimento do mercado associativo

## Drivers de Crescimento

1. **Sandbox cria legitimidade** — associações sairão da zona judicial. Mais fácil recrutar membros sem risco.
2. **Compliance obrigatório** → força adoção de software (hoje ~40% sem sistema adequado)
3. **Novas associações** — edital sandbox abrirá para novos projetos. Projeção: 315 → 600 (2028) → 800 (2030)
4. **Tendência regulatória global** — UY, ES, DE, NL já regulados. BR seguindo padrão europeu.
5. **Redução de barreiras** — OSS self-hosted elimina preocupação com dados em terceiros (barreira #1 hoje)

## FACT — Maior Deal Único

Federação das Associações de Cannabis Terapêutica:
- 36 associações membros
- 1 contrato federação = 36 instalações simultâneas
- Preço potencial: R$1.500/mês fixo federação + per-member por associação
- ARR potencial FACT deal: R$648k–R$1.2M (dependendo tamanho das associações)

## Premissas

- Average Revenue Per Association: R$450/mês M12 → R$590/mês M24 → R$780/mês M36 (upsell para planos maiores)
- Churn anual: 8% (associações que fecham ou migram para self-hosted)
- CAC: R$2.500 inicial (alto esforço de sales/onboarding) → R$800 quando produto-led growth via dossier template
- LTV/CAC: ~18× no M36 (R$780/mês × 12 meses / 0.08 churn-adj ÷ R$800 CAC)

## Comparável: Cannanas (Alemanha)

Cannanas atua no mercado alemão (KCanG 2024, modelo mais próximo ao BR):
- 600+ CSC = >50% do mercado DE
- €1/membro/mês
- Time ~30 pessoas, ARR estimado ~€7.2M
- Crescimento correlacionado com pressão regulatória (KCanG criou demand spurt idêntico ao que RDC 1.014 vai criar no BR)
