# canna-br

**OSS platform for Brazilian patient cannabis associations — ANVISA RDC 1.014/2026 sandbox.**

Self-hosted first. AGPL-3.0 + CLA. LGPD per-member crypto. SNGPC nativo. Zero vendor lock-in em dados de saúde.

## Gap Confirmado

Nenhum produto existente cobre simultaneamente:

- **LGPD** per-member AES-256-GCM + crypto-deletion (Art. 18 IV)
- **SNGPC XML** nativo (homologado ANVISA)
- **BSPO** trimestral/anual auto-gerado
- **CPC 29 / IAS 41** ativo biológico
- **Self-hosted** — associação é controladora única
- **OSS auditável** — código aberto = trust moat para dados de saúde em setor cinzento

## Arquitetura em Uma Frase

**Domain Kernel em TypeScript puro + [Emmett](https://github.com/event-driven-io/emmett) como event-sourcing kernel + raw para todo o resto.**

Domínio é função pura (`decide` / `evolve`). Emmett fornece event store, command handler e optimistic concurrency. HTTP/UI/DB schema ficam explícitos. Decisão em [ADR-001](/adr/0001-domain-kernel-emmett/).

## Stack

**Domain Kernel:** `TypeScript strict` · `Emmett` · `Vitest scenario coverage` · `ULID`

**Application:** `Fastify 5` · `Drizzle (read models)` · `BullMQ workers` · `PostgreSQL 16` · `Redis` · `MinIO` · `Kamal 2` · `TOTP sem SaaS auth`
