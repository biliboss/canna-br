Nota honesta — render inline ao vivo vs harness

member-quota-card foi capturado renderizando INLINE no chat ao vivo (iframe MCP App montado, dado real do Postgres). Os demais widgets aparecem renderizados com payload real via harness — porque o trigger por linguagem-natural sofre flake intermitente: o streaming de tool-args do AI SDK v6 + assistant-ui às vezes colapsa os argumentos para {} antes de chegar ao MCP (pior em tools multi-arg como dispensation). Os widgets e os tools estão corretos (bundles + handlers provados); o gargalo é a camada de streaming do host, rastreado para correção dedicada.
